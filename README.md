# ServiceNow Automated Employee Onboarding & Offboarding System

## 📌 Overview

This project automates the employee onboarding and offboarding process using ServiceNow. It ensures smooth employee lifecycle management and improves security.

## 🚀 Features

* Automated workflow
* Role-based access control
* HR, IT, Security and Facilities tasks
* Asset tracking
* Approvals and notifications

## 🛠 Tools

* ServiceNow Studio
* Flow Designer
* Kanban Board
* GitHub

## 📋 Modules

* Employee Table
* Onboarding Request
* Offboarding Request

## 🔄 Offboarding Workflow

1. Manager approval
2. HR approval
3. IT disable account
4. Security remove access
5. Asset return
6. Process complete
